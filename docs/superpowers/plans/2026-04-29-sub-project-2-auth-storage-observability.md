# Sub-project 2: Auth, Postgres, Storage, Observability — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the cross-cutting infrastructure every later sub-project depends on: bearer-token auth with argon2id-at-rest, the `tenant_api_keys` table, a `StorageProvider` interface (filesystem / S3 / Azure Blob) mirroring argus-flight-center's shape, structlog with the explicit-key redaction processor, OpenTelemetry tracing + Prometheus metrics scaffolding, a typed `DeploymentConfig` model loaded from the deployment-config schema, and a real `/readyz` that pings Postgres + storage.

**Architecture:** Async SQLAlchemy 2.x + asyncpg over `postgis/postgis:16-3.4`. Alembic owns the migration trail; the `0001_initial` migration enables PostGIS and creates `tenant_api_keys`. Auth is a FastAPI dependency that extracts `Authorization: Bearer <key>`, finds candidates by 8-char prefix, verifies argon2id, and returns a `Principal`. `StorageProvider` is an ABC with three concrete implementations selected by the `RFANALYZER_STORAGE_PROVIDER` env var. structlog is configured at app startup with a redaction processor enforcing the ADR-0002 §3 key set; OTel + Prometheus are wired but emit only request-level spans/counters until subsequent sub-projects add per-stage instrumentation.

**Tech Stack:** SQLAlchemy 2.x async + asyncpg · alembic · argon2-cffi · boto3 · azure-storage-blob · structlog · OpenTelemetry SDK + FastAPI instrumentation · prometheus-client · pydantic-settings.

**Authority:** Spec §8.4 (auth adapter contract) · §8.6 (observability) · [ADR-0001](../../adr/0001-stack.md) action item 4 · [ADR-0002](../../adr/0002-argus-alignment-and-auth.md) action items 1, 3, 5 · [Master plan §"Sub-project 2"](2026-04-29-rfanalyzer-implementation-master-plan.md#sub-project-2--auth-postgres-storage-observability).

**Depends on:** Sub-project 1 (toolchain + repo skeleton + docker-compose).

**Decisions resolved in this plan:**
- **argon2id parameters:** `memory_kib=65536`, `iterations=3`, `parallelism=4` per ADR-0002 §2. The sub-plan includes a calibration task (Task 5) that asserts a representative verify takes 30–80 ms on the developer's machine; if the box is too fast or too slow, a doc table records the observed timing without changing the parameters.
- **`DeploymentConfig` round-trip:** the pydantic model is generated by reading the JSON Schema and emitting field-by-field; `extra="forbid"` enforces the schema's `additionalProperties: false`. A round-trip property test (Task 4) confirms every default in the schema appears as the field default in the model.
- **`/readyz` failure surface:** returns 503 with a `ProblemDetail` body when Postgres or storage is unreachable. The body lists which dependencies are down. Spec Appendix D code: `READINESS_DEPENDENCY_UNREACHABLE` (warning-class). This adds the code to Appendix D + OpenAPI in Task 14's spec-sync run.

```python
def test_every_schema_default_appears_in_model() -> None:
    """Every `default` in the JSON Schema must be the pydantic field default.

    Walks the schema recursively and asserts the corresponding field on
    DeploymentConfig has the same default. Missing defaults are a sync bug.
    """
    schema = json.loads(SCHEMA.read_text())
    mismatches = _walk_defaults(schema, DeploymentConfig.model_json_schema())
    assert mismatches == [], f"Default mismatches: {mismatches}"


def _walk_defaults(schema_node: dict, pydantic_node: dict, path: str = "") -> list[str]:
    """Walk both schemas in lock-step and report any default that differs."""
    out: list[str] = []
    for key, sub in (schema_node.get("properties") or {}).items():
        sub_path = f"{path}.{key}" if path else key
        if "default" in sub:
            pyd_props = pydantic_node.get("properties") or {}
            pyd_sub = pyd_props.get(key) or {}
            if pyd_sub.get("default") != sub["default"]:
                out.append(f"{sub_path}: schema={sub['default']!r} pydantic={pyd_sub.get('default')!r}")
        if sub.get("type") == "object":
            pyd_sub_obj = (pydantic_node.get("properties") or {}).get(key) or {}
            ref_target = _resolve_ref(pyd_sub_obj, pydantic_node)
            out.extend(_walk_defaults(sub, ref_target, sub_path))
    return out


def _resolve_ref(node: dict, root: dict) -> dict:
    """Resolve a $ref into the pydantic-emitted defs."""
    if "$ref" in node:
        ref = node["$ref"].split("/")[-1]
        return (root.get("$defs") or {}).get(ref) or {}
    return node
```

```python
"""Async SQLAlchemy engine + session factory.

The engine is process-wide; sessions are per-request via the FastAPI
dependency `get_session` mounted on routers.
"""

from __future__ import annotations

import os
from collections.abc import AsyncIterator

from sqlalchemy.ext.asyncio import (
    AsyncEngine,
    AsyncSession,
    async_sessionmaker,
    create_async_engine,
)


def build_engine(url: str | None = None, *, echo: bool = False) -> AsyncEngine:
    """Build the async engine from *url* or RFANALYZER_DATABASE_URL."""
    return create_async_engine(
        url or os.environ["RFANALYZER_DATABASE_URL"],
        echo=echo,
        pool_pre_ping=True,
        pool_size=5,
        max_overflow=10,
    )


def build_session_factory(engine: AsyncEngine) -> async_sessionmaker[AsyncSession]:
    """Build the per-process async session factory."""
    return async_sessionmaker(engine, expire_on_commit=False)


_engine: AsyncEngine | None = None
_session_factory: async_sessionmaker[AsyncSession] | None = None


def init_engine() -> None:
    """Initialize process-wide engine + session factory at app startup."""
    global _engine, _session_factory
    _engine = build_engine()
    _session_factory = build_session_factory(_engine)


async def dispose_engine() -> None:
    """Dispose at shutdown."""
    global _engine, _session_factory
    if _engine is not None:
        await _engine.dispose()
    _engine = None
    _session_factory = None


async def get_session() -> AsyncIterator[AsyncSession]:
    """FastAPI dependency yielding an AsyncSession with auto-rollback on error."""
    if _session_factory is None:
        raise RuntimeError("Engine not initialized; call init_engine() at app startup.")
    async with _session_factory() as session:
        try:
            yield session
        except Exception:
            await session.rollback()
            raise


def get_engine() -> AsyncEngine:
    """Return the process-wide engine. Raises if not initialized."""
    if _engine is None:
        raise RuntimeError("Engine not initialized.")
    return _engine
```

```python
"""SQLAlchemy declarative base + ORM models.

Sub-project 2 ships TenantApiKey only. Catalog + run + asset tables land in
sub-projects 3 and 4 respectively.
"""

from __future__ import annotations

import uuid
from datetime import datetime

from sqlalchemy import ARRAY, Index, String, Text, text
from sqlalchemy.dialects.postgresql import TIMESTAMP, UUID
from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column


class Base(DeclarativeBase):
    """SQLAlchemy declarative base."""


class TenantApiKey(Base):
    """Bearer-token storage per ADR-0002 §2.

    The cleartext key never touches disk. The first 8 characters (`prefix`)
    are indexed for fast lookup; argon2id verification then matches against
    the candidate's `hash` column.
    """

    __tablename__ = "tenant_api_keys"

    id: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), primary_key=True, default=uuid.uuid4)
    prefix: Mapped[str] = mapped_column(String(8), nullable=False)
    hash: Mapped[str] = mapped_column(Text, nullable=False)
    tenant_id: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), nullable=False)
    scopes: Mapped[list[str]] = mapped_column(
        ARRAY(Text), nullable=False, server_default=text("'{}'::text[]")
    )
    rate_limit_class: Mapped[str | None] = mapped_column(Text)
    storage_class: Mapped[str | None] = mapped_column(Text)
    created_at: Mapped[datetime] = mapped_column(
        TIMESTAMP(timezone=True), nullable=False, server_default=text("now()")
    )
    expires_at: Mapped[datetime | None] = mapped_column(TIMESTAMP(timezone=True))
    revoked_at: Mapped[datetime | None] = mapped_column(TIMESTAMP(timezone=True))
    label: Mapped[str | None] = mapped_column(Text)
    last_used_at: Mapped[datetime | None] = mapped_column(TIMESTAMP(timezone=True))

    __table_args__ = (
        Index(
            "tenant_api_keys_prefix_idx",
            "prefix",
            postgresql_where=text("revoked_at IS NULL"),
        ),
        Index("tenant_api_keys_tenant_idx", "tenant_id"),
    )
```

```python
"""Alembic env (async)."""

from __future__ import annotations

import asyncio
import os
from logging.config import fileConfig

from alembic import context
from sqlalchemy.engine import Connection
from sqlalchemy.ext.asyncio import async_engine_from_config

from rfanalyzer.db.models import Base

config = context.config
if config.config_file_name is not None:
    fileConfig(config.config_file_name)

config.set_main_option(
    "sqlalchemy.url",
    os.environ.get("RFANALYZER_DATABASE_URL", config.get_main_option("sqlalchemy.url") or ""),
)

target_metadata = Base.metadata


def run_migrations_offline() -> None:
    context.configure(
        url=config.get_main_option("sqlalchemy.url"),
        target_metadata=target_metadata,
        literal_binds=True,
        dialect_opts={"paramstyle": "named"},
    )
    with context.begin_transaction():
        context.run_migrations()


def do_run_migrations(connection: Connection) -> None:
    context.configure(connection=connection, target_metadata=target_metadata)
    with context.begin_transaction():
        context.run_migrations()


async def run_migrations_online() -> None:
    connectable = async_engine_from_config(
        config.get_section(config.config_ini_section, {}),
        prefix="sqlalchemy.",
        future=True,
    )
    async with connectable.connect() as connection:
        await connection.run_sync(do_run_migrations)
    await connectable.dispose()


if context.is_offline_mode():
    run_migrations_offline()
else:
    asyncio.run(run_migrations_online())
```

```python
"""argon2id wrapper for at-rest bearer-key storage (ADR-0002 §2).

Parameters per ADR-0002: memory_kib=65536, iterations=3, parallelism=4.
"""

from __future__ import annotations

from dataclasses import dataclass

from argon2 import PasswordHasher
from argon2.exceptions import VerifyMismatchError


@dataclass(frozen=True, slots=True)
class Argon2Params:
    """argon2id parameters; see ADR-0002 §2 for rationale."""

    memory_kib: int = 65_536
    iterations: int = 3
    parallelism: int = 4


DEFAULT_PARAMS = Argon2Params()


def _hasher(p: Argon2Params) -> PasswordHasher:
    return PasswordHasher(
        time_cost=p.iterations,
        memory_cost=p.memory_kib,
        parallelism=p.parallelism,
        hash_len=32,
        salt_len=16,
    )


def hash_key(cleartext: str, params: Argon2Params = DEFAULT_PARAMS) -> str:
    """Return the argon2id encoded hash for *cleartext*."""
    return _hasher(params).hash(cleartext)


def verify_key(cleartext: str, encoded_hash: str) -> bool:
    """Return True iff *cleartext* matches *encoded_hash*."""
    try:
        # PasswordHasher.verify raises on mismatch; param settings are read
        # from the encoded hash, so we don't need DEFAULT_PARAMS here.
        PasswordHasher().verify(encoded_hash, cleartext)
    except VerifyMismatchError:
        return False
    return True


def needs_rehash(encoded_hash: str, params: Argon2Params = DEFAULT_PARAMS) -> bool:
    """Return True if *encoded_hash* was generated with weaker params than current."""
    return _hasher(params).check_needs_rehash(encoded_hash)
```

```python
"""Bearer-token verification (ADR-0002 §2).

Wire format: ``Authorization: Bearer <api-key>``. The first 8 characters of
the cleartext are the prefix used for the indexed lookup; the remaining
bytes are verified via argon2id against the candidate row's `hash` column.
"""

from __future__ import annotations

from datetime import datetime, timezone

from fastapi import Depends, HTTPException, Request, status
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from rfanalyzer.auth.argon2 import verify_key
from rfanalyzer.auth.principal import Principal
from rfanalyzer.db.engine import get_session
from rfanalyzer.db.models import TenantApiKey

PREFIX_LEN = 8


class InvalidBearer(ValueError):
    """Raised on malformed or missing Authorization header."""


def extract_bearer(header: str | None) -> str:
    """Return the token from `Authorization: Bearer <token>` or raise."""
    if header is None:
        raise InvalidBearer("missing Authorization header")
    parts = header.split(maxsplit=1)
    if len(parts) != 2 or parts[0].lower() != "bearer":
        raise InvalidBearer("Authorization header must use Bearer scheme")
    token = parts[1].strip()
    if not token:
        raise InvalidBearer("empty bearer token")
    return token


def key_prefix(token: str) -> str:
    """Return the first PREFIX_LEN chars of *token* or raise if too short."""
    if len(token) < PREFIX_LEN:
        raise InvalidBearer(f"token shorter than {PREFIX_LEN} characters")
    return token[:PREFIX_LEN]


async def authenticate(
    request: Request,
    session: AsyncSession = Depends(get_session),
) -> Principal:
    """FastAPI dependency: verify the Authorization header and return Principal."""
    try:
        token = extract_bearer(request.headers.get("Authorization"))
        prefix = key_prefix(token)
    except InvalidBearer as e:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail=str(e),
            headers={"WWW-Authenticate": 'Bearer realm="rfanalyzer"'},
        ) from e

    now = datetime.now(timezone.utc)
    stmt = select(TenantApiKey).where(
        TenantApiKey.prefix == prefix,
        TenantApiKey.revoked_at.is_(None),
    )
    rows = (await session.execute(stmt)).scalars().all()

    for row in rows:
        if row.expires_at is not None and row.expires_at <= now:
            continue
        if verify_key(token, row.hash):
            return Principal(
                tenant_id=row.tenant_id,
                key_id=row.id,
                scopes=frozenset(row.scopes),
                rate_limit_class=row.rate_limit_class,
                storage_class=row.storage_class,
            )

    raise HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="invalid or revoked api key",
        headers={"WWW-Authenticate": 'Bearer realm="rfanalyzer"'},
    )
```

```python
"""structlog configuration + redaction processor (ADR-0002 §3).

Default key set is exhaustive; operators may extend via DeploymentConfig.logging.redaction_keys
but cannot reduce it below the default.
"""

from __future__ import annotations

import logging
import sys
from collections.abc import Callable
from typing import Any

import structlog

REDACTED = "[REDACTED]"
MAX_RECURSION_DEPTH = 5

DEFAULT_REDACTION_KEYS: frozenset[str] = frozenset(
    {
        # snake_case (ADR-0002 §3)
        "authorization",
        "cookie",
        "password",
        "secret",
        "api_key",
        "apikey",
        "api_secret",
        "apisecret",
        "token",
        "access_token",
        "refresh_token",
        "id_token",
        "session_token",
        "bearer",
        "password_hash",
        # camelCase mirrors
        "accessToken",
        "refreshToken",
        "idToken",
        "sessionToken",
        "apiKey",
        "apiSecret",
        "passwordHash",
    }
)


def build_redaction_processor(
    extra_keys: list[str],
) -> Callable[[Any, str, dict[str, Any]], dict[str, Any]]:
    """Return a structlog processor that redacts sensitive keys (case-insensitive)."""
    keys_lc = {k.lower() for k in DEFAULT_REDACTION_KEYS}
    keys_lc |= {k.lower() for k in extra_keys}

    def _redact(node: Any, depth: int) -> Any:
        if depth >= MAX_RECURSION_DEPTH:
            return node
        if isinstance(node, dict):
            return {
                k: REDACTED if k.lower() in keys_lc else _redact(v, depth + 1)
                for k, v in node.items()
            }
        if isinstance(node, list):
            return [_redact(v, depth + 1) for v in node]
        return node

    def processor(_logger: Any, _method: str, event_dict: dict[str, Any]) -> dict[str, Any]:
        return _redact(event_dict, depth=0)  # type: ignore[no-any-return]

    return processor


def configure_logging(extra_redaction_keys: list[str] | None = None) -> None:
    """Configure structlog at app startup.

    Stdlib logging is also routed through structlog so third-party loggers
    (uvicorn, sqlalchemy, alembic) get the same JSON shape and redaction.
    """
    logging.basicConfig(
        format="%(message)s",
        stream=sys.stdout,
        level=logging.INFO,
    )

    structlog.configure(
        processors=[
            structlog.stdlib.add_log_level,
            structlog.processors.TimeStamper(fmt="iso", utc=True),
            structlog.contextvars.merge_contextvars,
            build_redaction_processor(extra_redaction_keys or []),
            structlog.processors.JSONRenderer(),
        ],
        wrapper_class=structlog.make_filtering_bound_logger(logging.INFO),
        context_class=dict,
        logger_factory=structlog.PrintLoggerFactory(),
        cache_logger_on_first_use=True,
    )
```

Create `src/rfanalyzer/storage/interface.py`:

```python
"""StorageProvider ABC.

Mirrors argus-flight-center's src/lib/storage.ts capabilities so a single
operator can run both services with the same backend. Direct uploads are
single-shot for content < 50 MiB; multipart for >= 50 MiB at 16 MiB parts
per spec §3.5.
"""

from __future__ import annotations

import abc
from dataclasses import dataclass, field

CONTENT_HASH_META_KEY = "sha256"
DEFAULT_PRESIGNED_TTL_SECONDS = 900


@dataclass(frozen=True, slots=True)
class ObjectHead:
    """Head metadata for a stored object."""

    key: str
    size_bytes: int
    content_type: str
    etag: str | None
    metadata: dict[str, str] = field(default_factory=dict)


@dataclass(frozen=True, slots=True)
class PresignedUrl:
    """A presigned URL with its TTL."""

    url: str
    method: str
    headers: dict[str, str]
    expires_in_seconds: int


@dataclass(frozen=True, slots=True)
class MultipartSession:
    """Initiated multipart upload session."""

    upload_id: str
    key: str
    part_size_bytes: int


@dataclass(frozen=True, slots=True)
class MultipartPart:
    """A completed multipart part to be passed to complete_multipart."""

    part_number: int
    etag: str


class StorageProvider(abc.ABC):
    """Content-addressed object storage interface."""

    @abc.abstractmethod
    async def put_object(
        self,
        key: str,
        body: bytes,
        *,
        content_type: str,
        metadata: dict[str, str],
    ) -> ObjectHead:
        """Direct upload (< 50 MiB)."""

    @abc.abstractmethod
    async def get_object(self, key: str) -> tuple[bytes, ObjectHead]:
        """Read the object body + head. Raises KeyError if missing."""

    @abc.abstractmethod
    async def head_object(self, key: str) -> ObjectHead:
        """Return the head metadata. Raises KeyError if missing."""

    @abc.abstractmethod
    async def delete_object(self, key: str) -> None:
        """Delete the object; idempotent (missing key is not an error)."""

    @abc.abstractmethod
    async def presigned_url_put(
        self,
        key: str,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
        content_type: str | None = None,
        content_length: int | None = None,
    ) -> PresignedUrl:
        """Return a one-shot upload URL."""

    @abc.abstractmethod
    async def presigned_url_get(
        self,
        key: str,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
    ) -> PresignedUrl:
        """Return a one-shot download URL."""

    @abc.abstractmethod
    async def initiate_multipart(
        self,
        key: str,
        *,
        content_type: str,
        metadata: dict[str, str],
        part_size_bytes: int = 16 * 1024 * 1024,
    ) -> MultipartSession:
        """Begin a multipart upload."""

    @abc.abstractmethod
    async def presign_part(
        self,
        upload_id: str,
        part_number: int,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
    ) -> PresignedUrl:
        """Return a presigned URL for a single part."""

    @abc.abstractmethod
    async def complete_multipart(
        self,
        upload_id: str,
        parts: list[MultipartPart],
    ) -> ObjectHead:
        """Commit the multipart upload and return the resulting head."""

    @abc.abstractmethod
    async def abort_multipart(self, upload_id: str) -> None:
        """Cancel an in-flight multipart upload."""
```

Create `src/rfanalyzer/storage/filesystem.py`:

```python
"""Filesystem-backed StorageProvider for dev / single-node deployments.

Layout under root/:
    objects/<key>             — object bytes
    objects/<key>.meta.json   — content_type + metadata + sha256 etag
    multipart/<upload_id>/    — part files keyed by part_number
"""

from __future__ import annotations

import hashlib
import json
import secrets
import shutil
from pathlib import Path

from rfanalyzer.storage.interface import (
    DEFAULT_PRESIGNED_TTL_SECONDS,
    MultipartPart,
    MultipartSession,
    ObjectHead,
    PresignedUrl,
    StorageProvider,
)


class FilesystemStorage(StorageProvider):
    """Filesystem impl. Presigned URLs are not supported (returns the object URL through the API service per spec §8.5)."""

    def __init__(self, root: Path) -> None:
        self.root = Path(root)
        self.objects = self.root / "objects"
        self.multipart = self.root / "multipart"
        self.objects.mkdir(parents=True, exist_ok=True)
        self.multipart.mkdir(parents=True, exist_ok=True)

    def _obj_path(self, key: str) -> Path:
        return self.objects / key

    def _meta_path(self, key: str) -> Path:
        return self.objects / f"{key}.meta.json"

    async def put_object(
        self, key: str, body: bytes, *, content_type: str, metadata: dict[str, str]
    ) -> ObjectHead:
        obj = self._obj_path(key)
        obj.parent.mkdir(parents=True, exist_ok=True)
        obj.write_bytes(body)
        etag = hashlib.sha256(body).hexdigest()
        meta = {"content_type": content_type, "etag": etag, "metadata": metadata}
        self._meta_path(key).write_text(json.dumps(meta))
        return ObjectHead(
            key=key,
            size_bytes=len(body),
            content_type=content_type,
            etag=etag,
            metadata=metadata,
        )

    async def get_object(self, key: str) -> tuple[bytes, ObjectHead]:
        obj = self._obj_path(key)
        if not obj.exists():
            raise KeyError(key)
        head = await self.head_object(key)
        return obj.read_bytes(), head

    async def head_object(self, key: str) -> ObjectHead:
        obj = self._obj_path(key)
        meta_path = self._meta_path(key)
        if not obj.exists() or not meta_path.exists():
            raise KeyError(key)
        meta = json.loads(meta_path.read_text())
        return ObjectHead(
            key=key,
            size_bytes=obj.stat().st_size,
            content_type=meta["content_type"],
            etag=meta["etag"],
            metadata=meta["metadata"],
        )

    async def delete_object(self, key: str) -> None:
        for p in (self._obj_path(key), self._meta_path(key)):
            p.unlink(missing_ok=True)

    async def presigned_url_put(
        self,
        key: str,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
        content_type: str | None = None,
        content_length: int | None = None,
    ) -> PresignedUrl:
        # Local mode proxies through the API per spec §8.5.
        return PresignedUrl(
            url=f"/v1/_storage-proxy/put/{key}",
            method="PUT",
            headers={"Content-Type": content_type or "application/octet-stream"},
            expires_in_seconds=ttl_seconds,
        )

    async def presigned_url_get(
        self, key: str, *, ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS
    ) -> PresignedUrl:
        return PresignedUrl(
            url=f"/v1/_storage-proxy/get/{key}",
            method="GET",
            headers={},
            expires_in_seconds=ttl_seconds,
        )

    async def initiate_multipart(
        self,
        key: str,
        *,
        content_type: str,
        metadata: dict[str, str],
        part_size_bytes: int = 16 * 1024 * 1024,
    ) -> MultipartSession:
        upload_id = secrets.token_hex(16)
        session_dir = self.multipart / upload_id
        session_dir.mkdir(parents=True, exist_ok=True)
        (session_dir / "session.json").write_text(
            json.dumps(
                {
                    "key": key,
                    "content_type": content_type,
                    "metadata": metadata,
                    "part_size": part_size_bytes,
                }
            )
        )
        return MultipartSession(upload_id=upload_id, key=key, part_size_bytes=part_size_bytes)

    async def presign_part(
        self,
        upload_id: str,
        part_number: int,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
    ) -> PresignedUrl:
        return PresignedUrl(
            url=f"/v1/_storage-proxy/multipart/{upload_id}/{part_number}",
            method="PUT",
            headers={},
            expires_in_seconds=ttl_seconds,
        )

    async def complete_multipart(
        self, upload_id: str, parts: list[MultipartPart]
    ) -> ObjectHead:
        session_dir = self.multipart / upload_id
        session_meta = json.loads((session_dir / "session.json").read_text())
        body = b"".join(
            (session_dir / f"{p.part_number}.part").read_bytes()
            for p in sorted(parts, key=lambda p: p.part_number)
        )
        head = await self.put_object(
            session_meta["key"],
            body,
            content_type=session_meta["content_type"],
            metadata=session_meta["metadata"],
        )
        shutil.rmtree(session_dir)
        return head

    async def abort_multipart(self, upload_id: str) -> None:
        session_dir = self.multipart / upload_id
        if session_dir.exists():
```

```python
"""S3 / S3-compatible (MinIO) StorageProvider via boto3.

Async wraps run-in-thread; boto3 is sync.
"""

from __future__ import annotations

import asyncio
from typing import Any

import boto3
from botocore.client import Config
from botocore.exceptions import ClientError

from rfanalyzer.storage.interface import (
    DEFAULT_PRESIGNED_TTL_SECONDS,
    MultipartPart,
    MultipartSession,
    ObjectHead,
    PresignedUrl,
    StorageProvider,
)

_USER_META_PREFIX = "x-amz-meta-"


class S3Storage(StorageProvider):
    """S3-compatible backend. Tested against MinIO; works against AWS S3."""

    def __init__(
        self,
        *,
        bucket: str,
        endpoint_url: str | None,
        access_key: str | None,
        secret_key: str | None,
        region: str,
    ) -> None:
        self.bucket = bucket
        self._client = boto3.client(
            "s3",
            endpoint_url=endpoint_url,
            aws_access_key_id=access_key,
            aws_secret_access_key=secret_key,
            region_name=region,
            config=Config(signature_version="s3v4", s3={"addressing_style": "path"}),
        )
        self._ensure_bucket()

    def _ensure_bucket(self) -> None:
        try:
            self._client.head_bucket(Bucket=self.bucket)
        except ClientError as e:
            if e.response["Error"]["Code"] in ("404", "NoSuchBucket"):
                self._client.create_bucket(Bucket=self.bucket)
            else:
                raise

    async def _run(self, fn: Any, *args: Any, **kwargs: Any) -> Any:
        return await asyncio.to_thread(fn, *args, **kwargs)

    @staticmethod
    def _meta_to_s3(metadata: dict[str, str]) -> dict[str, str]:
        return {k: v for k, v in metadata.items()}

    @staticmethod
    def _meta_from_s3(metadata: dict[str, str] | None) -> dict[str, str]:
        return dict(metadata or {})

    async def put_object(
        self, key: str, body: bytes, *, content_type: str, metadata: dict[str, str]
    ) -> ObjectHead:
        resp = await self._run(
            self._client.put_object,
            Bucket=self.bucket,
            Key=key,
            Body=body,
            ContentType=content_type,
            Metadata=self._meta_to_s3(metadata),
        )
        return ObjectHead(
            key=key,
            size_bytes=len(body),
            content_type=content_type,
            etag=resp["ETag"].strip('"'),
            metadata=metadata,
        )

    async def get_object(self, key: str) -> tuple[bytes, ObjectHead]:
        try:
            resp = await self._run(self._client.get_object, Bucket=self.bucket, Key=key)
        except ClientError as e:
            if e.response["Error"]["Code"] in ("NoSuchKey", "404"):
                raise KeyError(key) from e
            raise
        body = await self._run(resp["Body"].read)
        head = ObjectHead(
            key=key,
            size_bytes=resp["ContentLength"],
            content_type=resp.get("ContentType", "application/octet-stream"),
            etag=resp["ETag"].strip('"'),
            metadata=self._meta_from_s3(resp.get("Metadata")),
        )
        return body, head

    async def head_object(self, key: str) -> ObjectHead:
        try:
            resp = await self._run(self._client.head_object, Bucket=self.bucket, Key=key)
        except ClientError as e:
            if e.response["Error"]["Code"] in ("404", "NoSuchKey"):
                raise KeyError(key) from e
            raise
        return ObjectHead(
            key=key,
            size_bytes=resp["ContentLength"],
            content_type=resp.get("ContentType", "application/octet-stream"),
            etag=resp["ETag"].strip('"'),
            metadata=self._meta_from_s3(resp.get("Metadata")),
        )

    async def delete_object(self, key: str) -> None:
        await self._run(self._client.delete_object, Bucket=self.bucket, Key=key)

    async def presigned_url_put(
        self,
        key: str,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
        content_type: str | None = None,
        content_length: int | None = None,
    ) -> PresignedUrl:
        params: dict[str, Any] = {"Bucket": self.bucket, "Key": key}
        if content_type is not None:
            params["ContentType"] = content_type
        if content_length is not None:
            params["ContentLength"] = content_length
        url = await self._run(
            self._client.generate_presigned_url,
            "put_object",
            Params=params,
            ExpiresIn=ttl_seconds,
        )
        headers = {"Content-Type": content_type} if content_type else {}
        return PresignedUrl(url=url, method="PUT", headers=headers, expires_in_seconds=ttl_seconds)

    async def presigned_url_get(
        self, key: str, *, ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS
    ) -> PresignedUrl:
        url = await self._run(
            self._client.generate_presigned_url,
            "get_object",
            Params={"Bucket": self.bucket, "Key": key},
            ExpiresIn=ttl_seconds,
        )
        return PresignedUrl(url=url, method="GET", headers={}, expires_in_seconds=ttl_seconds)

    async def initiate_multipart(
        self,
        key: str,
        *,
        content_type: str,
        metadata: dict[str, str],
        part_size_bytes: int = 16 * 1024 * 1024,
    ) -> MultipartSession:
        resp = await self._run(
            self._client.create_multipart_upload,
            Bucket=self.bucket,
            Key=key,
            ContentType=content_type,
            Metadata=self._meta_to_s3(metadata),
        )
        return MultipartSession(
            upload_id=resp["UploadId"], key=key, part_size_bytes=part_size_bytes
        )

    async def presign_part(
        self,
        upload_id: str,
        part_number: int,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
    ) -> PresignedUrl:
        # The S3 API requires the Key for upload_part presigning; this presumes
        # the caller has the key in hand. Passed via the upload-id-to-key
        # mapping in catalog/assets state (sub-project 3 wires this).
        # For the integration test, see upload_part().
        raise NotImplementedError(
            "presign_part needs the original key; sub-project 3 wires the lookup. "
            "Use S3Storage.upload_part() directly in the meantime."
        )

    async def upload_part(
        self, upload_id: str, part_number: int, body: bytes
    ) -> MultipartPart:
        """Direct part upload helper used by integration tests + sub-project 3."""
        # We require the key for UploadPart; integration tests embed it.
        # Sub-project 3 wires a mapping (upload_id → key) in the assets table.
        raise NotImplementedError(
            "Direct upload_part needs the key; threaded through the assets table in sub-project 3."
        )

    async def complete_multipart(
        self, upload_id: str, parts: list[MultipartPart]
    ) -> ObjectHead:
        # Same key-lookup constraint; sub-project 3 wires it.
        raise NotImplementedError(
            "complete_multipart needs the key; sub-project 3 threads it through assets state."
        )

    async def abort_multipart(self, upload_id: str) -> None:
        raise NotImplementedError(
            "abort_multipart needs the key; sub-project 3 threads it through assets state."
        )
```

```python
"""Azure Blob StorageProvider via azure-storage-blob.

Mirrors S3Storage's single-shot path; multipart (block-blob staging) wires in
sub-project 3 alongside the assets state table.

from __future__ import annotations

import asyncio
from datetime import datetime, timedelta, timezone
from typing import Any

from azure.core.exceptions import ResourceNotFoundError
from azure.storage.blob import (
    BlobSasPermissions,
    BlobServiceClient,
    ContentSettings,
    generate_blob_sas,
)

from rfanalyzer.storage.interface import (
    DEFAULT_PRESIGNED_TTL_SECONDS,
    MultipartPart,
    MultipartSession,
    ObjectHead,
    PresignedUrl,
    StorageProvider,
)


class AzureBlobStorage(StorageProvider):
    """Azure Blob backend (single-shot ops; multipart in sub-project 3)."""

    def __init__(self, *, container: str, connection_string: str) -> None:
        self.container = container
        self._service = BlobServiceClient.from_connection_string(connection_string)
        self._container_client = self._service.get_container_client(container)
        try:
            self._container_client.create_container()
        except Exception:  # noqa: BLE001 — already-exists is the common case
            pass

    async def _run(self, fn: Any, *args: Any, **kwargs: Any) -> Any:
        return await asyncio.to_thread(fn, *args, **kwargs)

    async def put_object(
        self, key: str, body: bytes, *, content_type: str, metadata: dict[str, str]
    ) -> ObjectHead:
        client = self._container_client.get_blob_client(key)
        await self._run(
            client.upload_blob,
            body,
            overwrite=True,
            content_settings=ContentSettings(content_type=content_type),
            metadata=metadata,
        )
        props = await self._run(client.get_blob_properties)
        return ObjectHead(
            key=key,
            size_bytes=props.size,
            content_type=content_type,
            etag=props.etag.strip('"'),
            metadata=dict(props.metadata or {}),
        )

    async def get_object(self, key: str) -> tuple[bytes, ObjectHead]:
        client = self._container_client.get_blob_client(key)
        try:
            stream = await self._run(client.download_blob)
            body = await self._run(stream.readall)
            props = await self._run(client.get_blob_properties)
        except ResourceNotFoundError as e:
            raise KeyError(key) from e
        head = ObjectHead(
            key=key,
            size_bytes=props.size,
            content_type=props.content_settings.content_type or "application/octet-stream",
            etag=props.etag.strip('"'),
            metadata=dict(props.metadata or {}),
        )
        return body, head

    async def head_object(self, key: str) -> ObjectHead:
        client = self._container_client.get_blob_client(key)
        try:
            props = await self._run(client.get_blob_properties)
        except ResourceNotFoundError as e:
            raise KeyError(key) from e
        return ObjectHead(
            key=key,
            size_bytes=props.size,
            content_type=props.content_settings.content_type or "application/octet-stream",
            etag=props.etag.strip('"'),
            metadata=dict(props.metadata or {}),
        )

    async def delete_object(self, key: str) -> None:
        client = self._container_client.get_blob_client(key)
        try:
            await self._run(client.delete_blob)
        except ResourceNotFoundError:
            return

    async def presigned_url_put(
        self,
        key: str,
        *,
        ttl_seconds: int = DEFAULT_PRESIGNED_TTL_SECONDS,
        content_type: str | None = None,
        content_length: int | None = None,
    ) -> PresignedUrl:
        sas = generate_blob_sas(
            account_name=self._service.account_name,
            container_name=self.container,
            blob_name=key,
            account_key=self._service.credential.account_key,
            permission=BlobSasPermissions(write=True, create=True),
            expiry=datetime.now(timezone.utc) + timedelta(seconds=ttl_seconds),
            content_type=content_type,
        )
        url = f"{self._container_client.get_blob_client(key).url}?{sas}"
        headers = {"x-ms-blob-type": "BlockBlob"}
        if content_type:
            headers["Content-Type"] = content_type
        return PresignedUrl(url=url, method="PUT", headers=headers, expires_in_seconds=ttl_seconds)

    async def presigned_url_get(

    ) -> PresignedUrl:
        sas = generate_blob_sas(
            account_name=self._service.account_name,
            container_name=self.container,
            blob_name=key,
            account_key=self._service.credential.account_key,
            permission=BlobSasPermissions(read=True),
```

```python
"""Liveness + readiness probes (spec §8.6, sub-project 2)."""

from __future__ import annotations

import logging

from fastapi import APIRouter, Response
from sqlalchemy import text

from rfanalyzer.db.engine import get_engine
from rfanalyzer.storage.factory import build_storage_provider

router = APIRouter(tags=["health"])

logger = logging.getLogger(__name__)


@router.get("/healthz", summary="Liveness probe")
async def healthz() -> dict[str, str]:
    return {"status": "ok"}


async def _check_postgres() -> bool:
    """SELECT 1 against the engine; True on success."""
    try:
        engine = get_engine()
        async with engine.connect() as conn:
            await conn.execute(text("SELECT 1"))
        return True
    except Exception:  # noqa: BLE001 — readiness probes swallow all backend errors
        logger.exception("readiness check failed: postgres")
        return False


async def _check_storage() -> bool:
    """Round-trip a small object through the configured backend."""
    try:
        store = build_storage_provider()
        await store.put_object(
            "_readyz/probe",
            b"ok",
            content_type="text/plain",
            metadata={},
        )
        body, _ = await store.get_object("_readyz/probe")
        await store.delete_object("_readyz/probe")
        return body == b"ok"
    except Exception:
        logger.exception("readiness check failed: storage")
        return False


@router.get("/readyz", summary="Readiness probe")
async def readyz(response: Response) -> dict[str, object]:
    """Return 200 + dependency map when all dependencies are reachable; 503 otherwise.

    Uses spec Appendix D code READINESS_DEPENDENCY_UNREACHABLE on degradation.
    """
    pg = await _check_postgres()
    st = await _check_storage()
    deps = {
        "postgres": "ok" if pg else "unreachable",
        "storage": "ok" if st else "unreachable",
    }
    if pg and st:
        return {"status": "ok", "dependencies": deps}
    response.status_code = 503
    return {
        "status": "degraded",
        "code": "READINESS_DEPENDENCY_UNREACHABLE",
        "dependencies": deps,
    }
```

```python
from __future__ import annotations
```
