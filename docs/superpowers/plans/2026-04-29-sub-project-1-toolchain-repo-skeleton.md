# Sub-project 1: Toolchain & Repo Skeleton — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the empty RfAnalyzer repo as a working Python project with green CI, a Docker Compose dev stack (api + worker + postgis + minio), and the OpenAPI emission diff gate — no runtime behavior beyond `/healthz` and `/readyz` stubs.

**Architecture:** A `src/`-layout Python 3.12 package managed by `uv`, exposing a FastAPI app via `uvicorn` and a separate worker process. Every subsystem under `src/rfanalyzer/` is created as a stub module with a docstring citing its spec section, locking the layout so sub-projects 2–6 only fill bodies. CI runs lint → typecheck → unit → integration → openapi-diff → fuzz on every PR; docker-compose pins `postgis/postgis:16-3.4` per ADR-0002 and a MinIO sidecar for the eventual `StorageProvider` S3 backend.

**Tech Stack:** Python 3.12 · uv · FastAPI · uvicorn · pydantic v2 · pydantic-settings · structlog · OpenTelemetry · ruff · mypy · pytest + pytest-asyncio · hypothesis · Schemathesis · GitHub Actions · Docker · Docker Compose · `postgis/postgis:16-3.4` · `minio/minio`.

**Authority:** [ADR-0001](../../adr/0001-stack.md) action items 1–3, 5; [ADR-0002](../../adr/0002-argus-alignment-and-auth.md) action item 2 (compose pin + PostGIS extension boot); [master plan §"Sub-project 1"](2026-04-29-rfanalyzer-implementation-master-plan.md#sub-project-1--toolchain--repo-skeleton).

**Decisions resolved in this plan:**
- **`src/rfanalyzer/_generated/openapi.yaml` is committed**, not gitignored (master plan open question #3). PR diffs surface OpenAPI changes for review.
- **The dep set in `pyproject.toml` is the frozen v1 dep set.** Later sub-projects import against these pinned versions and do not add new top-level dependencies without explicit revision of this plan.
- **Worker is a separate Python process** (`python -m rfanalyzer.runs.worker`) with a stub claim loop that sleeps. Real claim logic lands in sub-project 4.

```python
"""Liveness and readiness endpoints (spec §8.6).

Sub-project 1 ships /healthz returning 200 unconditionally and /readyz
returning 200 unconditionally. Sub-project 2 wires real Postgres + storage
reachability checks into /readyz.
"""

from typing import Any

from fastapi import APIRouter

router = APIRouter(tags=["health"])


@router.get("/healthz", summary="Liveness probe")
async def healthz() -> dict[str, Any]:
    """Return 200 unconditionally; the process is alive."""
    return {"status": "ok"}


@router.get("/readyz", summary="Readiness probe")
async def readyz() -> dict[str, Any]:
    """Return 200 when dependencies are reachable.

    Sub-project 1 returns 200 unconditionally; sub-project 2 adds Postgres +
    storage reachability checks per ADR-0002.
    """
    return {"status": "ok"}
```

```python
"""FastAPI application factory.

Sub-project 1 wires only the health router. Subsequent sub-projects mount
catalog, asset, run, analysis, and webhook routers.
"""

from fastapi import FastAPI

from rfanalyzer import __version__
from rfanalyzer.api import health


def create_app() -> FastAPI:
    """Build and return a configured FastAPI application."""
    app = FastAPI(
        title="RfAnalyzer",
        version=__version__,
        description=(
            "Self-hosted, single-tenant API for RF propagation analysis. "
            "Spec under docs/superpowers/specs/."
        ),
    )
    app.include_router(health.router)
    return app
```

```python
"""Worker process: claims SUBMITTED runs from Postgres via SKIP LOCKED (spec §8.1).

Sub-project 1 ships a stub that polls once per `poll_interval_seconds` and
does nothing. Sub-project 4 implements the real claim loop:

    SELECT … FROM runs WHERE status = 'SUBMITTED'
    FOR UPDATE SKIP LOCKED LIMIT 1;

with lease + lease_token + tile-write idempotence per ADR-0001 / spec §8.1.
"""

from __future__ import annotations

import asyncio
import logging

logger = logging.getLogger(__name__)

DEFAULT_POLL_INTERVAL_SECONDS = 5.0


async def claim_one_iteration() -> None:
    """Claim and process one Run, or return None if no SUBMITTED run is waiting.

    Sub-project 1 stub: always returns None. Sub-project 4 implements the real
    SKIP-LOCKED claim against the runs table.
    """
    return None


async def run_worker_loop(poll_interval_seconds: float = DEFAULT_POLL_INTERVAL_SECONDS) -> None:
    """Run the worker poll loop until cancelled."""
    logger.info("rfanalyzer.worker.starting poll_interval_seconds=%s", poll_interval_seconds)
    try:
        while True:
            await claim_one_iteration()
            await asyncio.sleep(poll_interval_seconds)
    except asyncio.CancelledError:
        logger.info("rfanalyzer.worker.stopping")
        raise
```

Create `scripts/diff-openapi.py`:

```python
#!/usr/bin/env python3
"""Diff the emitted (pydantic-derived) OpenAPI against the spec-derived OpenAPI.

Run: ``python scripts/diff-openapi.py --emitted PATH --spec PATH``

Sub-project 1 only enforces that both files are structurally valid YAML +
both declare an OpenAPI 3.x version. Sub-project 3 lights up real component-
schema diffing once entity models exist.

Exit code 0 = no drift; nonzero = drift (printed to stdout).
"""

from __future__ import annotations

import argparse
import sys
from pathlib import Path
from typing import Any

import yaml

REPO_ROOT = Path(__file__).resolve().parents[1]
DEFAULT_EMITTED = REPO_ROOT / "src" / "rfanalyzer" / "_generated" / "openapi.yaml"
DEFAULT_SPEC = REPO_ROOT / "docs" / "superpowers" / "specs" / "2026-04-25-rf-site-planning-api.openapi.yaml"


def load(path: Path) -> dict[str, Any]:
    """Load a YAML file and assert it declares an OpenAPI 3.x version."""
    data = yaml.safe_load(path.read_text())
    if not isinstance(data, dict):
        raise SystemExit(f"{path}: top-level YAML must be a mapping")
    version = data.get("openapi", "")
    if not (isinstance(version, str) and version.startswith("3.")):
        raise SystemExit(f"{path}: missing or invalid openapi: 3.x declaration")
    return data


def diff(emitted: dict[str, Any], spec: dict[str, Any]) -> list[str]:
    """Return a list of drift findings.

    Sub-project 1: empty list (both files are structurally valid). Sub-project 3
    will compare components.schemas keys + structure. Sub-project 6 widens to
    paths + operations.
    """
    return []


def main() -> int:
    """CLI entry point. Return 0 on no drift, 1 on drift."""
    parser = argparse.ArgumentParser(description=__doc__)
    parser.add_argument("--emitted", type=Path, default=DEFAULT_EMITTED)
    parser.add_argument("--spec", type=Path, default=DEFAULT_SPEC)
    args = parser.parse_args()

    emitted_doc = load(args.emitted)
    spec_doc = load(args.spec)
    findings = diff(emitted_doc, spec_doc)
    if findings:
        print("OpenAPI drift detected:")
        for f in findings:
            print(f"  - {f}")
        return 1
    print(f"OK: {args.emitted} structurally valid; {args.spec} structurally valid; no drift findings.")
    return 0


if __name__ == "__main__":
    sys.exit(main())
```
